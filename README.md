# Slider Scroll

A modern, responsive HTML template featuring a fullscreen menu and image slider components.

## Features

- **Responsive Design**: Adapts to all screen sizes
- **Fullscreen Menu**: Interactive navigation menu
- **Image Slider**: Smooth image carousel with navigation controls
- **Modern Typography**: Uses Fahkwang and Poppins fonts
- **Custom Animations**: Smooth transitions and interactions
- **Clean Code**: Well-structured HTML and CSS

## Technical Details

- **Framework**: Webflow
- **CSS Preprocessor**: Not used (vanilla CSS)
- **Responsive Breakpoints**:
  - Desktop: 992px and up
  - Tablet: 768px - 991px
  - Mobile: 479px - 767px
  - Tiny: Up to 478px

## File Structure

- `page.html` - Main HTML file containing all content and structure
- Embedded CSS - All styles are included in the HTML head section
- Webflow Icons - Custom icon font for UI elements

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile Safari (iOS)
- Chrome for Android

## Usage

1. Open `page.html` in a web browser to view the template
2. Customize the content directly in the HTML file
3. Modify styles in the `<style>` section of the head
4. Update images by replacing the source files in the `images` directory

## Customization

### Colors
- Primary Background: #e9c6dd
- Text Color: #0d0628
- Accent Colors: Can be modified in the CSS variables

### Typography
- Headings: Fahkwang, sans-serif
- Body Text: Poppins, sans-serif

## Notes

- This template uses Webflow's custom classes and JavaScript
- All animations are handled with CSS transitions
- The menu and slider components are fully responsive
